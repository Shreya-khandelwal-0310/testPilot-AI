# AI-Powered Automated Testing Tool 🚀

An AI-powered automated testing platform built using React.js, Node.js, Express.js, and Selenium WebDriver.

This project automates UI testing workflows, generates dynamic test cases, and executes browser automation using Selenium.

---

## Features

✅ Dynamic test case generation  
✅ Automated UI testing using Selenium  
✅ Frontend-triggered automation workflows  
✅ REST API integration  
✅ Chrome browser automation  
✅ Screenshot capture during automation  
✅ End-to-end login automation testing  
✅ Full-stack architecture using React and Node.js

## Automated Testing Flow

- Open browser automatically
- Navigate to login page
- Enter username and password
- Click login button
- Validate successful login
- Capture screenshot
- Close browser automatically

## Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Shreya-khandelwal-0310/testPilot-AI.git
```

---

### Backend Setup

```bash
cd backend
npm install
npx nodemon server.js
```

---

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## Run Application

Open browser:

http://localhost:3000

---

## Selenium Automation

Click:

Run Login Automation

Automation will:

- Open Chrome browser
- Execute login automation
- Capture screenshot
- Close browser automatically
